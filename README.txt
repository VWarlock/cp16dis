The Western Digital CP16xx/WD21xx chipset was used in:
* DEC PDP-11/03 minicomputer
* Western Digital WD16 chipset (used in Alpha Micro AM100)
* Wester Digital WD9000 chipset (Pascal Microengine)

This Python 3 program is a disassembler for CP16xx/WD21xx microcode.

At present, it does not produce output that can be assembled with
any known assembler or microassembler.
